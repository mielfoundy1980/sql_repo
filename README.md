# sql_repo : Data Warehouse and Analytics Project
This is my first SQL Project: From developping a data warehouse using SQL Server to generating business and actionable insights.

### Specifications
- **Data Sources**: Import data from two source systems (ERP & CRM) provided as CSV Files.
- **Data Quality**: Cleanse and resolve data quality issues prior to analysis.
- **Integration**: Combine both sources into a single data model.

### Business Intelligence : Analytics & Reporting
#### Objective
Develop SQL based analytics to deliver detailed insights about:
- **Customer Behavior**
- **Product Performance**
- **Sales Trends**

**Note: This work is a part of my learning journey in data engineering and data analytics. All the content is inspired from a Youtube Channel (Data With Baraa)**
